# reyamira

**Shipping software (sometimes).**

A space to build, fix, ship, and explore. Open-source by default, made for people.
Run by [Ari Mayer](https://reyamira.com).

## What we ship

- **[pass-cli](https://github.com/reyamira/pass-cli)** — Password and API-key vault for the terminal. AES-256-GCM, BIP39 recovery, system-keychain integration. CLI + TUI.
- **[models](https://github.com/reyamira/models)** — TUI for AI models, benchmarks, and provider statuses.
- **[aic](https://github.com/reyamira/aic)** — Latest changelogs for AI coding assistants — Claude Code, Codex, Gemini CLI, Copilot CLI, OpenCode, and more.

Install: `brew install reyamira/tap/<name>` · `scoop bucket add reyamira https://github.com/reyamira/scoop-bucket`

More at **[reyamira.com](https://reyamira.com)**.
